Definitions

HTML stands for HyperText Markup Language.
HyperText is text displayed on a computer or device that provides access to other text through links, also known as hyperlinks.
Markup Language is a computer language that defines the structure and presentation of raw text.
Articles <article> specifies independent, self-contained content such as forum, blog, or news story posts.
Paragraphs <p> contain a block of plain text.
Spans <span> contain short pieces of text or even individual words.
Short Quotations <q> contain shorter quotes in a text. It will be surrounded by quotation marks.
Long Quotations <blockquote> contain longer quotes in a text. It will be surrounded by quotation marks.
Asides <aside> contain content aside from the content it is placed in and should be indirectly related to the surrounding content such as a sidebar.
Abbreviations <abbr> reprepresent abbreviations or acronyms.

CSS Selectors, Chaining, and Descendant Combinator

Universal Selector example: * {color: black;}
Type Selector example: type {color: maroon;}
Class Selector example = .class {color: red;}
ID Selector example = #id {color: deeppink;}
Multiple Selector example = type, .class, #id {color: purple;}
Style Selector example = strong {color: indigo;}
Attribute Selector example = a[href*='example'] {color: blue;}
Pseudo-Class Selector example = a:hover {color: cyan;}
Chaining example = type.class {color: lightgreen;}
Descendant Combinator example = .class type {color: green;}
Another Example for tables = .class li {color: greenyellow;}

CSS Positioning

Static (Default)
  • Positions element according to normal flow of document.
Relative
  • Positions element relative to it's default static position.
  • Top, Right, Bottom, Left
Absolute
  • Positions element relative to it's closest positioned partent.
  • Ignored by other elements
  • Scrolls with document
  • Top, Right, Bottom, Left
Fixed
  • Positions element relative to the viewport.
  • Ignored by other elements
  • Does not scroll with document
  • Top, Right, Bottom, Left
Sticky
  • Positions element relative to nearest scrolling ancestor and containing block.
  • Ignored by other elements
  • Does not scroll with document until offset properties are met
  • Creates 'stacking context'
  • Top, Right, Bottom, Left
